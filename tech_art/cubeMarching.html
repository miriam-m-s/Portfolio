Cube Marching — Technical Overview

I made this terrarin usin the Marching Cubes algorithm, which enabled real-time terrain destruction. I also developed the click-based interaction system (clicker), which triggered the visual effects (VFX) on spell impact. Additionally, I created the terrain shader, which dynamically responded to changes such as material transitions and surface deformation with RVT and vertexColor.

Features

  • Procedural generation using Perlin noise .
  • Dynamic destruction of terrain and vegetation.
